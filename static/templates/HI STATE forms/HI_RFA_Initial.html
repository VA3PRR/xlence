Hawaii RFA (Request for Assistance)
x Request for Assistance Form Instructions

  1. Incident name: name of the incident or event (it will be saved for the next time the form is used).
  2. To: The name/position of the person this form is to be finally delivered by hand.
  3. From: the originator of the form contents.
  4. Date: current date.
  5. Time: current time.
  6. Priority: select the proper priority of this request.
  7. Date/Time Needed: the date/time the request should be delivered.
  8. Point of Contact: name of individual that can answer questions about the request.
  9. Date/Time Release: the date/time the asset requested can be released.
  10. POC email: the email address of the point of contact above.
  11. Ph: phone number of the POC.
  12. County: the county that the location of the delivery resides.
  13. Requester's Organization: name of organization sending the request.
  14. Location Coordinates: the latitude and longitude of the delivery site.
    Please use 4 decimal digits right of decimal pt.
  15. Description: a concise description of what is needed and why.
  16. Remarks: any additional helpful information that could expedite the request.
  17. Delivery address: the location to deliver the requested materials.
  18. Site POC: the name of contact at the delivery location.
  19. Site Ph: the phone number of the site POC.
  20. Delivery instructions: any additional info to aid delivery, such as gate code or additional contacts.
  21. Approved by, position / title: required name and position of the approving personnel.
